# SaaS Roadmap

## Phase 1 — MVP

- User auth
- Credits
- Upload person/product images
- Consent checkbox
- 5 ad styles
- Local PIL compositor
- Download result
- History
- Stripe placeholder

## Phase 2 — Paid launch

- Stripe webhook to upgrade plan and add credits
- Watermark removal for paid plans
- Cloud image storage
- PostgreSQL
- Email verification
- Password reset
- Admin dashboard

## Phase 3 — AI improvement

- Background removal with rembg or paid segmentation API
- Vision model captions for person/product
- Better product cutout detection
- Multiple ad variations per generation
- Template marketplace
- Brand kit upload

## Phase 4 — Agency tools

- Batch product upload
- Team accounts
- Client workspaces
- White-label export
- API access
